# EventON

EventON is a college-focused app designed to streamline the organization of events such as workshops, competitions, and talk sessions. It features QR code-generated tickets for easy check-ins, payment monitoring, and estimation of activity points for participants. The app fully digitizes the event management process, making it more efficient, user-friendly, and paperless.
